Creating an Ext4 Filesystem Image File
======================================

Follow the steps below on computer running Linux to create an image file
formatted using the Ext4 filesystem. 

1. In any convenient location, create a new directory named Ext4FilesystemImage.

2. Open a command prompt window within the newly created directory.

3. Create an empty file of the proper size by running the following command:

	dd if=/dev/zero of=mkfs.img bs=4k count=1024

4. Format the image file as an Ext4 filesystem by running the following command:

	sudo mkfs.ext4 

5. Mount the image file as an accessible filesystem by running the following command:

	mount ext4.img ext4/
